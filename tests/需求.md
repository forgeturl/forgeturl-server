实现多个基于request、pytest-allure库的python3的http接口测试脚本，用于测试这个服务器的接口相关功能，脚本存放于forgeturl-server/app/tests/下。
一、背景知识如下：
接口定义在forgeturl-server/app/api/proto下的多个.proto文件中。
若请求头包含X-Token: test，则标识该请求是一个登录态请求，否则为非登录态请求。test是测试使用的token，实际用户会有自己的token。

二、测试用例如下：
```bash
1. 非登录态，可以拉取到用户信息和页面信息
# 获取用户信息
curl '127.0.0.1:80/space/getUserInfo' -d '{"uid": 1}' -H 'content-type: application/json'

# 获取页面数据
curl '127.0.0.1:80/space/getPage' -d '{"page_id": "test_page_id"}' -H 'content-type: application/json'

2. 登录态，可以拉取详细用户信息
curl '127.0.0.1:80/space/getUserInfo' -d '{"uid": 1}' -H 'content-type: application/json' -H 'X-Token: test'

3. 登录态，拉取我的空间
curl '127.0.0.1:80/space/getMySpace' -d '{"uid": 1}' -H 'content-type: application/json' -H 'X-Token: test'

4. 非登录态，创建页面失败，得到错误码code != 1的返回。登录态，创建页面，创建的页面id在返回json的page_id字段里。
curl '127.0.0.1:80/space/createPage' -d '{
  "title": "我的页面",
  "brief": "这是一个测试页面",
  "collections": [
    {
      "links": [
        {
          "title": "示例链接",
          "url": "https://example.com",
          "tags": ["示例", "测试"],
          "photo_url": "https://example.com/photo.jpg",
          "sub_links": [
            {
              "sub_title": "子链接",
              "sub_url": "https://sub.example.com"
            }
          ]
        }
      ]
    }
  ]
}' -H 'content-type: application/json' -H 'X-Token: test'

5. 非登录态，无法拉取getMySpace，得到错误码code != 1的返回。。登录态拉取我的空间，确认上一步创建的页面在返回的列表里。
curl '127.0.0.1:80/space/getMySpace' -d '{"uid": 1}' -H 'content-type: application/json' -H 'X-Token: test'

6. 非登录态，拉取不到上一部创建的页面，得到错误码code != 1的返回。登录态，拉取上一步创建的页面
curl '127.0.0.1:80/space/getPage' -d '{"page_id": "test_page_id"}' -H 'content-type: application/json' -H 'X-Token: test'

7. 非登录态，更新页面失败，得到错误码code != 1的返回。登录态，更新上一步创建的页面
curl '127.0.0.1:80/space/updatePage' -d '{
  "page_id": "O3sFmpq",
  "title": "更新后的页面标题",
  "brief": "更新后的页面描述",
  "collections": [
    {
      "links": [
        {
          "title": "更新后的链接",
          "url": "https://updated.example.com",
          "tags": ["更新", "测试"]
        }
      ]
    }
  ],
  "version": 0,
  "mask": 7
}' -H 'content-type: application/json' -H 'X-Token: test'

8. 非登录态，创建链接失败，得到错误码code != 1的返回。登录态，创建页面链接
curl '127.0.0.1:80/space/addPageLink' -d '{
  "page_id": "test_page_id",
  "page_type": "readonly"
}' -H 'content-type: application/json' -H 'X-Token: test'

9. 非登录态，移除页面链接失败，得到错误码code != 1的返回。登录态，移除页面链接
curl '127.0.0.1:80/space/removePageLink' -d '{"page_id": "test_page_id"}' -H 'content-type: application/json' -H 'X-Token: test'

10. 非登录态，删除页面失败，得到错误码code != 1的返回。登录态，删除页面
curl '127.0.0.1:80/space/deletePage' -d '{"page_id": "test_page_id"}' -H 'content-type: application/json' -H 'X-Token: test'

11. 非登录态，调整页面ID顺序失败，得到错误码code != 1的返回。登录态，调整页面ID顺序
curl '127.0.0.1:80/space/savePageIds' -d '{
  "uid": 1,
  "page_ids": ["page2", "page1", "page3"]
}' -H 'content-type: application/json' -H 'X-Token: test'

```
